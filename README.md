# MyTranslator
MyTranslator is an application, that translates words from english to russian using Skyeng Dictionary API.

Main features:
- Translation search
- Translation card view
- History view
- Search in history

Technologies used:
- MVVM
- App modularization
- Kotlin DSL
- Coroutines
- Koin
- Android Jetpack Navigation
- Retrofit
- Room
- Coil
- SplashScreen API
- Skyeng Dictionary API
